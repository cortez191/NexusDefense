Nexus Defense 0.30 PUBLIC BETA
Fanowska gra typu TowerDefense w uniwersum League Of Legends


Gra jest w pełni darmowa, wszystkie modele postaci, tekstury jak i dźwięki należą do firmy RiotGames.

Gra nie jest jeszcze w pełni zoptymalizowana, więc mogą się zdarzyć spadki fps-ów nawet na komputerach spełniających zalecane wymagania.

Dodatkowe informacje:

Menu opcji jest wyłączone, natomiast ręcznie można ustawiać jakość modeli przeciwników jak i wieżyczek, domyślnie jest ustawione
na 1 (ustawienia niskie) - ze względu na problemy z płynnością i braku zaimplementowanej optymalizacji.

W katalogu Data znajduje się plik config.cfg. W nim można zmienić następujące parametry

Config.TurretsQuality = 1 <- ustawienia normalne wiezyczek
Config.TurretsQuality = 2 <- ustawienia wysokie wiezyczek  (-50% fps)
Config.TurretsQuality = 3 <- ustawienia ultra wiezyczek (-100% fps)

Config.EnemiesQuality = 1 <- ustawienia normalne przeciwników
Config.EnemiesQuality = 2 <- ustawienia wysoki przeciwników (-50% fps)
Config.EnemiesQuality = 3 <- ustawienia ultra przeciwników	(-50% fps)

Nie zalecamy zmieniać ustawień na wyższe niż 1 na słabszych komputerach

Config.ShadowQuality = 0  <- obecnie nie działa

Minimalne wymagania:

Procesor:		Dwurdzeniowy procesor Intel Core 2 Duo 2.0Ghz lub lepszy
Pamięć Ram:		512 MB Ram
Karta Graficzna:		Intel HD 3000, NVIDIA Geforce 520, AMD Radeon 5450 lub lepsza (karta musi być zgodna z OpenGL 3.3)
Miejsce na dysku:		1GB


Zalecane wymagania:

Procesor:		Czterordzeniowy procesor lub lepszy
Pamięć Ram:		1GB Ram
Karta Graficzna:		Intel HD 4000, NVIDIA Geforce 630, AMD Radeon 5570 lub lepsza (karta musi być zgodna z OpenGL 3.3)
Miejsce na dysku:		2GB
